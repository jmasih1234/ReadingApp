Instructions

Setup Instructions for Researchers

1. Clone the Repository

git clone <repository-url>
cd ReadingApp

2. Install Django

Make sure you have Python installed on your machine, then install Django:

pip install django

3. Create a Virtual Environment (Recommended)

A virtual environment helps isolate project dependencies:

# Create virtual environment
python -m venv venv

# Activate virtual environment
# Windows:
venv\Scripts\activate

# macOS/Linux:
source venv/bin/activate

4. Install Requirements

pip install -r requirements.txt

5. Navigate to Project Directory

cd ReadingApp
cd ReadingApp

6. Run Database Migrations

python manage.py migrate

7. Create a Superuser (Optional, for Admin Access)

python manage.py createsuperuser

8. Start the Development Server

python manage.py runserver

The application will be available at http://127.0.0.1:8000/

Using the Application

For Researchers

  1. Configure Conditions: Set up the 8 different reading conditions by arranging them in the sequence slots on the home page.
  2. Assign Participant ID: Enter a unique participant identifier (e.g., P001, P002) for tracking purposes.
  3. Begin Trial: Click "Begin trial" to start the experiment sequence. Each condition will be applied to 3 text sections automatically.
  4. Monitor Progress: The application will cycle through all 24 text passages (8 conditions × 3 sections each).
  5. Access Data: After completion, access the admin panel at http://127.0.0.1:8000/admin/ to view and export trial logs and survey data.

For Participants

  1. Start Reading: When a text passage appears, click the "Start" button and begin reading at your normal pace.
  2. Finish Reading: When you complete the passage, immediately click the "Finished Reading" button.
  3. Repeat: Continue this process for all 24 text passages. The visual conditions will change automatically according to the pre-set sequence.
  4. Complete Survey: After reading all passages, you'll be presented with a post-trial survey. Drag and drop to rank the 8 reading conditions from most preferred (top) to least preferred (bottom).
  5. Submit: Click "Submit Survey" to complete the experiment.

Data Collection

The application automatically tracks and stores:

  • Participant ID
  • Reading condition (font size, colors, font family)
  • Section index
  • Start and end timestamps
  • Reading duration in milliseconds
  • Word count
  • Words per minute (WPM) — calculated automatically
  • Preference rankings from the post-trial survey
