About ReadingApp

ReadingApp is a Django-based web application designed to support HCI research focused on understanding how visual design variables affect reading performance and comprehension. This Web App helps explore how visual design variables impact reading performance.

Problem Statement

In HCI, the way text appears onscreen (Text color/background color, text size, font, etc.) can affect how difficult it is for a person to read the text. Suboptimal choices in formatting can slow reading speed, impede comprehension, and increase cognitive load, which may impact efficiency and user satisfaction. People read text on screens every day on websites, apps, e-books, learning platforms, and more, so text design choices are greatly important to people. Specifically students use screens to read text daily for classes, so we are focusing on college students as our age demographic to help them with readability in their daily lives with college.

People interact with digital text daily through websites, apps, e-books, and learning platforms, making text design choices critically important. College students, in particular, rely on screens to read text for classes, assignments, and research. Despite typography being a key element of interface design, there is not yet a consensus among researchers on how specific visual factors directly affect reading speed and comprehension.

Studying this research problem is essential because it provides evidence-based insights for designers, developers, and educators to make informed design decisions that optimize readability for students and the general population. By systematically testing the effects of these variables, this research contributes to creating more accessible, efficient, and user-friendly digital reading experiences.

Primary Goals & Objectives

The primary goal of this research is to understand how visual design variables, specifically text and background color combinations, font size, and font styles affect reading performance and comprehension in everyday human-computer interactions.

As computer science students, we frequently encounter text in multiple environments:

  • Essays written in strict academic formatting (MLA, Chicago, APA)
  • Code displayed in Integrated Development Environments (IDEs) with different themes and color schemes
  • Reading assignments on digital platforms with varied defaults

Although formatting guidelines exist for academic work, they are typically designed for consistency rather than readability optimization. Likewise, IDEs and digital platforms often allow customization of text appearance, but there is limited research-based guidance on which visual factors actually improve reading speed and comprehension for the average user.

This study aims to address that gap by systematically testing how font size, text/background color combinations, and font family influence two dependent variables:

  • Reading speed — measured in words per minute (WPM)
  • User satisfaction — gauged from a post-trial preference survey

How This Application Helps

ReadingApp provides a controlled environment for conducting reading performance experiments. Researchers can configure different visual conditions (font sizes, colors, font families) and have participants read text passages under each condition. The application automatically tracks:

  • Reading duration for each text passage
  • Words per minute (WPM) calculations
  • Participant preferences through a ranking survey
  • All data is logged to a database for later analysis

By collecting quantitative performance metrics alongside qualitative preference data, this tool enables researchers to identify which visual design choices truly optimize reading performance and align with user preferences.
